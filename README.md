# Meme-Genrator-
This is a simple React application that allows users to generate memes by choosing a random meme image from the Imgflip API and customizing the top and bottom text.
# Features
Fetches meme templates from an external API

Lets users add custom top and bottom text

Displays a new meme image on button click

# Technologies Used
React (useState, useEffect)

JavaScript (ES6+)

HTML/CSS
